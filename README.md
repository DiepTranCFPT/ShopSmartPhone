# HSF Smartphone Shop

A web application for managing and shopping smartphones, built with Java, Spring Boot, Thymeleaf, and Maven.

## Features

- Product listing and detail view
- Add to cart and buy now functionality
- User authentication and role-based access
- Responsive UI with Bootstrap

## Technologies

- Java 17+
- Spring Boot
- Maven
- Thymeleaf
- Bootstrap 5

## Getting Started

### Prerequisites

- JDK 17 or newer
- Maven 3.6+
- SQL Server database

### Setup

1. Clone the repository: git clone https://github.com/DiepTranCFPT/ShopSmartPhone.git
2. Configure your database in `src/main/resources/application.properties`.
3. Build and run the project:mvn spring-boot:run
4. Access the app at [http://localhost:8080](http://localhost:8080)
